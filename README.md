
# STYLE STORE

Style Store is an E-commerce application for clothing.

[STYLE STORE : Live Link](https://stylestore.netlify.app/)

## This includes:

- Home page
- Login page
- Sign Up Page
- Product Listing Page
- Cart Management
- Wishlist Management
